## Estratégia Recomendada para o Projeto

Para um projeto que precisa de contexto de lesões/desfalques antes das partidas:

### Fluxo ideal de coleta

1. **Antes de cada rodada** → buscar artigo no `ge.globo.com/cartola` com suspensos + lesionados + escalações prováveis (artigo único cobre todos os 20 times)
2. **Confirmar** → cruzar com `flashscore.com.br/noticias` que também publica artigo consolidado por rodada
3. **Jogadores individuais** → usar `sofascore.com` para status individual (disponível/duvidoso/lesionado)
4. **Lesões de longo prazo** → `transfermarkt.com.br` para histórico e estimativa de retorno

### Padrão das URLs úteis para scraping

```
# GE Globo Cartola (suspensos + lesionados por rodada)
https://ge.globo.com/cartola/noticia/{ANO}/{MES}/{DIA}/cartola-{ANO}-veja-suspensos-lesionados-e-escalacoes-provaveis-da-rodada-{N}-do-brasileirao.ghtml

# Flashscore (desfalques e escalações por rodada)
https://www.flashscore.com.br/noticias/futebol-brasileirao-betano-confira-desfalques-e-escalacoes-provaveis-da-{N}-rodada-do-brasileirao/{ID}/

# SofaScore (status do jogador)
https://www.sofascore.com/pt/time/futebol/{CLUBE}/{ID}

# Transfermarkt (lesões do clube)
https://www.transfermarkt.com.br/{clube}/verletztespieler/verein/{ID}
```
